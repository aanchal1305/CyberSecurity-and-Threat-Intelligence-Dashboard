Overview:

This Power BI dashboard provides a comprehensive single-page intelligence view of global cybersecurity threats spanning 3,000 confirmed incidents across 10 nations over a 10-year period (2015–2024). Built as a CIA/CBI-grade analytical tool, it is designed for business analysts, security teams, and executive stakeholders who need to move from raw incident data to actionable intelligence in a single briefing.

The dashboard answers seven sequential intelligence questions:
Who is attacking → Where the damage lands → Why breaches succeed → Which weapons are used → Whom is targeted → Whether it is escalating → What is working to stop it
